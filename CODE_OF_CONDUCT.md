# Codigo de conducta

## Compromiso

0E3 busca mantener espacios de trabajo profesionales, respetuosos y seguros para colaboradores, clientes y comunidad.

## Comportamientos esperados

- Comunicacion clara y respetuosa.
- Feedback tecnico enfocado en el problema.
- Cuidado con informacion sensible de clientes, usuarios y sistemas.
- Colaboracion responsable en issues, pull requests y documentacion.

## Comportamientos no aceptados

- Ataques personales, acoso o discriminacion.
- Publicacion de datos privados o credenciales.
- Uso de informacion interna fuera del contexto autorizado.
- Conductas que comprometan la seguridad del proyecto o terceros.
